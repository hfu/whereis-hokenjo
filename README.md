# whereis-hokenjo
# usage
```zsh
$ rake -T                                                                                                                                 [~/whereis-hokenjo]
rake build:style     # build style
rake build:tiles     # build tiles
rake copy:mapbox_gl  # copy files from mapbox-gl-js
rake host            # host the site
```
